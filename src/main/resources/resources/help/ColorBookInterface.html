Coloring Book Mode

At the end of each case after the done button has been clicked, you will enter Coloring Book Mode. During this mode, you will be asked to reassess things you have done by coloring nodes (findings, hypotheses, etc) different colors based on whether you think they are correct. Color nodes you think are correct green, and nodes you are unsure about yellow.


To color nodes:

  1. Left click the color button you would like to use to select it,

  2. Left click the nodes you would like to color that particular color. You can recolor nodes a different color by selecting a different color and clicking the node.


Alternatively, you can color nodes by right clicking them and selecting the appropriate color from the choices. You must color all nodes before moving on to the next case. After you are done coloring your nodes, click the button.
